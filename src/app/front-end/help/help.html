Quick Start Guide

Welcome to the Minecraft Local Server Manager! Follow these steps to get started:

  1. Create a new server by clicking "Add Server" on the Server List page.
  2. Configure your server settings, including game version and world type.
  3. Start your server from the Workspace page.
  4. Connect to your server using the local IP address and port number provided.

Frequently Asked Questions

How do I install plugins on my server?
To install plugins, navigate to the "Plugins" tab in your server's workspace. You can upload .jar files directly or browse our curated list of popular plugins. After uploading, restart your server to activate the plugins.
How can I make my server accessible from the internet?
To make your server accessible from the internet, you'll need to set up port forwarding on your router. Navigate to your router's configuration page and forward the port your Minecraft server is using (default is 25565) to your local PC's IP address.
How do I change the world seed?
To change the world seed, stop your server, go to the "Settings" tab in your server's workspace, and look for the "World Seed" option. Enter your desired seed and save the changes. When you start the server again, a new world will be generated with the new seed.
Can I run multiple servers at the same time?
Yes, you can run multiple servers simultaneously. Each server will need its own unique port number. Be mindful of your computer's resources, as running multiple servers can be demanding on CPU and RAM.

Troubleshooting

Common Issues:

  • Server won't start: Check your Java installation and ensure you have allocated enough RAM.
  • Can't connect to server: Verify the server is running and check your firewall settings.
  • Lag or poor performance: Lower your render distance and limit the number of players or plugins.

For more detailed troubleshooting, please consult our comprehensive guide.
